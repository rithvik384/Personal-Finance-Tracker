# Personal Finance Tracker

A console-based C++ application to track income and expenses, calculate totals, and sort transactions by category.

## Features
- Add income and expenses with category and amount
- View all transactions
- View summary: total income, total expenses, and balance
- Sort transactions by category

## How to Run
1. Compile the program:
   g++ main.cpp -o FinanceTracker
2. Run the program:
   ./FinanceTracker

## Skills and Concepts
- C++: vectors, structs, functions, console I/O
- Sorting and dynamic data handling
- Problem-solving and logic implementation
